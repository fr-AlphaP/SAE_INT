Présentation du projet

La Tanière Fertile est une ferme expérimentale connectée, développée par des étudiants de l’IUT Réseaux & Télécoms. Ce site permet au fermier et à ses acolytes de :

  • 📊 Surveiller les données en temps réel via Grafana
  • 💡 Contrôler les équipements à distance (effaroucheur, lampe, clim...)
  • 📍 Localiser les dispositifs sur une carte
  • 📲 Recevoir des alertes via un bot Telegram

Explorez les différentes sections via la navigation ci-dessus.

🧠 Comment fonctionne notre système

Notre ferme connectée repose sur une architecture moderne composée des briques suivantes :

  • Capteurs LoRaWAN : transmettent les données environnementales (bruit, température...)
  • MQTT : protocole léger pour publier les données
  • Telegraf : collecte et redirige vers InfluxDB
  • InfluxDB : stockage temporel
  • Grafana : visualisation des données
  • Bot Telegram : envoie des alertes automatiquement
  1. LoRaWAN → MQTT
  2. MQTT → Telegraf
  3. Telegraf → InfluxDB
  4. InfluxDB → Grafana
  5. Grafana → Bot Telegram

Toutes ces briques travaillent ensemble pour une gestion fluide et accessible depuis n'importe quel navigateur.
